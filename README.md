# ludicrousspeed
To conflict with Lightspeed AI

## Goal
Use the `c` executable to take input similar to lightspeed, but to return a URL to Google (or similar engine), that just searches for the query you requested.

## Possible Flags

| Flag | Possible Use Case |
| ---- | ----------------- | 
| -b | Bing |
| -d | DuckDuckGo |
| -g | Google (default) |
| -v | verbose |
| -y | Expresses opinions on Yogurt. |
